# ScarletTrails
Scarlet Trails Team Git

An Android application created by Kheiyasa Carter, Rafaela da Fonseca, Scott Millspaugh, Vincius Ito Nagura, and Nathan Stemmler for Software Engineering (CSC 380) at SUNY Oswego during the Fall 2014 semester.

An application that provides the user with information and location services for trails in Oswego County while providing social interactions through user submitted photos, comments, and reviews.
